# Todo Api

This an Api for Tasks with Authorization

## Installation

```
git clone https://github.com/learningcampili/mern-todo-backend.git
```

```
npm  install
```

## Create Mongodb

```
docker compose up -d
```

## Before Running App

```
rename .env.example to .env
```

complete .env variables

## Run App

```
npm run dev
```
